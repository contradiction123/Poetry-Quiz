诗词答题自动化工具 - 使用说明
================================

一、首次使用
1. 解压文件到任意目录
2. 将 config.example.json 重命名为 config.json
3. 用记事本打开 config.json，填入你的 DeepSeek API Key
   （在 "api_key" 字段中填入你的密钥）

二、运行程序
1. 双击 PoetryAnswerTool.exe 启动程序
2. 首次运行会自动下载OCR模型（约100MB），请保持网络连接
3. 等待模型下载完成后，程序界面会自动打开

三、配置说明
- API Key获取：访问 https://platform.deepseek.com/ 注册并获取API Key
- 配置文件：config.json（程序会自动创建，首次需要手动配置API Key）
- 数据库：answer_history.db（自动创建，记录答题历史）

四、使用步骤
1. 选择游戏模式（诗词答题 或 材料匹配）
2. 点击"设置区域"按钮，框选游戏区域（可选，默认全屏）
3. 点击"开始运行"按钮开始自动化
4. 程序会自动识别题目并点击正确答案

五、常见问题
1. 如果程序无法启动，请检查是否被杀毒软件拦截
2. 如果OCR识别不准确，可以调整"设置区域"框选更精确的区域
3. 如果程序运行缓慢，可以关闭其他占用资源的程序

六、技术支持
如遇问题，请检查：
- config.json 中的 API Key 是否正确配置
- 网络连接是否正常（首次运行需要下载模型）
- 防火墙是否允许程序访问网络
